# 🤝 加入 SynergyHub

感谢你对 SynergyHub 感兴趣！我们欢迎所有形式的贡献。

## 🎯 如何贡献

### 报告 Bug
1. 前往 [Issues](https://github.com/7190246-bit/SynergyHub/issues)
2. 选择 Bug Report 模板
3. 描述问题 + 复现步骤

### 提出新功能
1. 前往 [Issues](https://github.com/7190246-bit/SynergyHub/issues)
2. 选择 Feature Request
3. 描述你的需求和使用场景

### 提交代码
1. Fork 项目
2. 创建分支：`git checkout -b feature/your-feature`
3. 编写代码 + 测试
4. 提交：`git commit -m 'Add xxx'`
5. 推送：`git push origin feature/your-feature`
6. 提交 Pull Request

## 📋 开发规范

- 使用 Python 3.8+
- 遵循 PEP 8
- 添加必要的注释
- 提交前测试

## 💬 交流方式

- GitHub Issues：问题反馈
- GitHub Discussions：交流讨论
