## 🎯 核心场景

### 内容生产团队
```
内容策划 → AI 写作 → 人工审核 → 自动发布
```
1 个运营 + 3 个 Agent = 10 人内容团队产出

### 客户服务
```
智能分流 → 销售 Agent → 技术 Agent → 满意度回访
```
7×24 小时响应，处理 80% 常见问题

### 数据分析
```
数据采集 → 清洗分析 → 生成报告 → 自动推送
```
从"每周报表"升级为"实时洞察"

### 快速开始

```bash
# 安装
pip install synergyhub

# 运行示例
python -m synergyhub.examples.content_team

# 或体验多 Agent 协作
python -m synergyhub.examples.customer_service
```
